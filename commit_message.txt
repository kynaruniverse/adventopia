Feat: Implement PWA and landscape-only overhaul for Adventopia

This commit introduces a comprehensive overhaul to transform Adventopia into a Progressive Web App (PWA) optimized for landscape-only orientation across all devices. Key changes include:

- **PWA Manifest (`manifest.json`):** Updated to enforce `standalone` display mode and `landscape` orientation.
- **Service Worker (`service-worker.js`):** Enhanced caching strategy to include all game assets for robust offline play and faster loading. Cache version incremented to `adventopia-v4`.
- **HTML Structure (`index.html`):** Added an `orientation-warning` div to guide users in portrait mode, along with meta tags for improved PWA integration and JavaScript logic for orientation detection and PWA installation prompts.
- **CSS Styling (`style.css`):** Refactored to prioritize landscape layout and ensure full responsiveness across mobile, tablet, and desktop screens. New styles for the orientation warning have been added.
- **Core JavaScript (`main.js`):** Integrated orientation detection and locking logic, including `checkOrientation()` calls on load, resize, and orientation change events. The `elements` object now includes a reference to the `orientation-warning` element.

These changes collectively aim to provide a more immersive, accessible, and performant gaming experience for Adventopia users.
